### <p align="center">E-Bucket</p>

#### <p align="center"> An E-Commerce App Clone Made In Flutter With Firebase</p>

#### <p align="center"> **E-Bucket** is a modern and user-friendly e-commerce app clone built using **Flutter** and **Firebase**. It supports essential e-commerce features such as user authentication, product listings, a shopping cart, and order management.

## Features:

* **User Authentication**: Sign up, log in, and manage user sessions with Firebase Authentication.
* **Product Listings**: Display products stored in Firebase Firestore, with categories and search functionality.
* **Shopping Cart**: Add and remove items, view total prices, and proceed to checkout.
* **Order Management**: Place orders, view order history, and track order status.

## Tech Stack:

* **Frontend**: Flutter
* **Backend**: Firebase (Firestore, Authentication, Cloud Storage)
* **Storage**: Firebase Storage (for storing product images)
